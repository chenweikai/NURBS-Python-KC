Control Points Manager
^^^^^^^^^^^^^^^^^^^^^^

The ``control_points`` module provides helper functions for managing control points. It is a better alternative to
the :doc:`compatibility module <module_compatibility>` for managing control points. Please refer to the following
class references for more details.

* :class:`.control_points.CurveManager`
* :class:`.control_points.SurfaceManager`
* :class:`.control_points.VolumeManager`
