Voxelization
^^^^^^^^^^^^

.. versionadded:: 5.0

``voxelize`` module provides functions for voxelizing NURBS volumes. :py:func:`voxelize` also supports multi-threaded
operations via `multiporcessing <https://docs.python.org/2/library/multiprocessing.html>`_ module.
