Freeform Geometry
^^^^^^^^^^^^^^^^^

.. versionadded:: 5.2

``freeform`` module provides classes for representing freeform geometry objects.

`Freeform` class provides a basis for storing freeform geometries. The points of the geometry can be set via the
`evaluate()` method using a keyword argument.
